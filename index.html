The protocol stack you're building toward
MCP
Agent ↔ Tools layer
Anthropic / Linux Foundation. 97M downloads. De facto standard for tool access. You consume MCP servers — you don't build them.
A2A
Agent ↔ Agent layer
Google / Linux Foundation. Cross-framework agent coordination via Agent Cards + JSON-RPC 2.0. Native in Bedrock AgentCore.
AG-UI
Agent ↔ UI layer
Open / CopilotKit. Event-based streaming of agent state to frontends in real-time. Native in Bedrock AgentCore since Mar 2026.
ACP
Agentic Commerce layer
OpenAI + Stripe. Commerce checkout protocol (live Sep 2025). Domain-specific — not a general agent comms protocol. Know it; don't build for it in year one.
