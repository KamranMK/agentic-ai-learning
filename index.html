North star
Understand architectures and approaches to building agentic AI systems — not collect framework certifications. Every project answers a specific architectural question and touches a real protocol layer.
The protocol stack you're building toward
MCP
Agent ↔ Tools layer
Anthropic / Linux Foundation. 97M downloads. De facto standard for tool access. You consume MCP servers — you don't build them.
A2A
Agent ↔ Agent layer
Google / Linux Foundation. Cross-framework agent coordination via Agent Cards + JSON-RPC 2.0. Native in Bedrock AgentCore.
AG-UI
Agent ↔ UI layer
Open / CopilotKit. Event-based streaming of agent state to frontends in real-time. Native in Bedrock AgentCore since Mar 2026.
ACP
Agentic Commerce layer
OpenAI + Stripe. Commerce checkout protocol (live Sep 2025). Domain-specific — not a general agent comms protocol. Know it; don't build for it in year one.

⬡ Architecture insight
PydanticAI's type system caught 23 bugs in a 90-day benchmark that would have hit production in LangChain. Strands cuts setup from ~40 lines to ~3 for the 80% case. One LangGraph practitioner described it as "a genuine surprise." The architectural question: what does each abstraction layer cost you in control, and when does that cost actually matter in production?

⬡ Architecture insight
The full stack: MCP connects agents to tools. A2A connects agents to each other. AG-UI connects agents to users. Most engineers know MCP. Engineers who can deploy a production system spanning all three layers in AgentCore — with proper observability and guardrails — are genuinely rare in 2026. That's your target position.

When to reach for which framework
PydanticAISingle agents or small typed graphs. Validated outputs critical. Multi-LLM provider setup. Code quality over ecosystem. Default choice for structured agent logic with Claude. Docs · GitHub
StrandsAWS-native deployment. Model-driven, no hardcoded workflows. Native A2A + MCP + multi-agent patterns. 14M+ downloads; used in Amazon Q Developer internally. Primary choice for Bedrock production. Docs · GitHub
LangGraphComplex conditional branching. Stateful workflows with checkpointing. Human-in-the-loop at specific nodes. Long-term maintainable systems. Complements Strands. Docs
CrewAIRapid prototyping. Role-based agent teams. Speed to demo over control. Good for POCs — not for production systems requiring long-term security compliance. Docs
AutoGen/AG2Research and experimentation only. Near-zero security mechanisms. Any 2024 article recommending it for enterprise production predates the AG2 split — treat with caution. Docs
